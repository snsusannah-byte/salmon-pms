# 进口单证通知版式设计（微信聊天适用）

> 纯文本格式，可直接复制粘贴到微信聊天窗口

---

## 1️⃣ 预计到货通知

```
【预计到货通知】

合同编号：8691
空运单号：020-04893173

到货日期：2026-04-24
宰杀日期：2026-04-22

规格箱数：
6-7kg：67箱
7-8kg：41箱
9kg+：22箱
合计：130箱
```

**设计要点**:
- 纯文本，无表格
- 无总金额、无重量
- 每行一个信息，换行清晰
- 可直接复制粘贴到微信

---

## 2️⃣ 预计税金通知

```
【预计税金通知】

发票号：8691
预计税金：¥30,257.22
```

**设计要点**:
- 只显示最终结果
- 不展示计算过程
- 一行发票号，一行税金金额
